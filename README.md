# Resume Writing Agent Skills

Research-backed agent skills for writing resumes that actually get interviews. Every rule traces back to a real study, not recycled blog advice.

## Skills

| Skill | What it covers |
| --- | --- |
| `resume-writing` | Bullet composition, word choice, keyword placement, tailoring, gendered language bias |
| `resumx` | Resumx Markdown-to-PDF renderer syntax, CLI, tags, views, page fitting, multi-language |
